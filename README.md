# HolyMeal v0.4

## Описание проекта
HolyMeal - генератор планов питания и списков покупок с использованием AI (Claude API)

## Основные компоненты

### Модели данных
1. **Meal** - модель рецепта:
   - Поля: название, ингредиенты, инструкции, время приготовления, калорийность, тип блюда
   - Методы:
     - from_dict/to_dict - преобразование в/из словаря
     - get_main_ingredients - получение основных ингредиентов

2. **Ingredient** - модель ингредиента:
   - Поля: название, количество, единица измерения

3. **ShoppingList** - модель списка покупок:
   - Методы:
     - add_meal - добавление рецепта с учетом множителя дней
     - get_sorted_items - получение отсортированного списка
     - clear - очистка списка

### Сервисы
1. **MealService** - основной сервис для работы с рецептами:
   - Генерация плана питания
   - Регенерация отдельных рецептов
   - Валидация и проверка уникальности рецептов
   - Интеграция с Claude API

2. **PromptService** - генерация промптов для AI

3. **ShoppingService** - работа со списками покупок

### Маршруты
1. **meal_routes.py** - маршруты для работы с рецептами
2. **shopping_routes.py** - маршруты для работы со списками покупок

### Валидация
1. **MealValidator** - валидация данных рецептов

### Логирование
1. **logger.py** - настройка и управление логами

### Шаблоны
1. Основные:
   - index.html - главная страница
   - shopping_list.html - страница списка покупок

2. Ошибки:
   - 404.html - страница не найдена
   - 500.html - внутренняя ошибка сервера

## Зависимости
- Python 3.10+
- Flask
- Anthropic (Claude API)
- Другие зависимости указаны в requirements.txt

## Конфигурация
Настройки хранятся в config/settings.py:
- CLAUDE_MODEL - модель Claude
- CLAUDE_MAX_TOKENS_TO_SAMPLE - максимальное количество токенов
- CLAUDE_TEMPERATURE - параметр температуры для генерации
- DEFAULT_PEOPLE_COUNT - количество порций по умолчанию


## 🚀 Запуск проекта

Перед запуском убедитесь, что у вас установлен Python ≥ 3.11 и пакетный менеджер [`uv`](https://github.com/astral-sh/uv).

### 1. Установить зависимости

Установите все зависимости из `pyproject.toml`:

```bash
uv sync
````

---

### 2. Запустить приложение

Запуск основного файла приложения (`main.py`) через `uv`:

```bash
uv run main.py
```

### 3. Открыть в браузере

После запуска приложение будет доступно по адресу:

```bash
http://localhost:5000
```

Откройте в браузере вручную или запустите команду:

```bash
open http://localhost:5000  # macOS
```
