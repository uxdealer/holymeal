# Техническая документация HolyMeal v0.4

## 1. Общее описание проекта

HolyMeal - это веб-приложение для генерации планов питания и списков покупок с использованием искусственного интеллекта (Claude API). Приложение позволяет создавать персонализированные планы питания с учетом предпочтений пользователя, доступных ингредиентов и диетических рекомендаций.

### Основные возможности:
- Генерация планов питания на неделю (будни и выходные)
- Создание списков покупок на основе рецептов
- Регенерация отдельных блюд
- Редактирование ингредиентов
- Учет доступных ингредиентов
- Расчет порций для нескольких человек

## 2. Архитектура приложения

Приложение построено на основе микросервисной архитектуры с использованием Flask в качестве веб-фреймворка. 

### Основные компоненты архитектуры:
- **Frontend**: HTML, CSS, JavaScript (чистый JS без фреймворков)
- **Backend**: Python/Flask
- **AI Integration**: Claude API (Anthropic)
- **Хранение данных**: In-memory + LocalStorage (браузер)

### Структура проекта:
```
holymeal/
├── app/
│   ├── models/          # Модели данных
│   ├── routes/          # Маршруты API
```

## 4. API Endpoints

### Meal Routes (app/routes/meal_routes.py)

#### POST /generate
Генерирует полный план питания.
```json
Request:
{
    "availableIngredients": ["ингредиент1", "ингредиент2"]
}

Response:
{
    "meal_plan": [
        {
            "name": "название блюда",
            "ingredients": ["ингредиент - количество"],
            "instructions": ["шаг1", "шаг2"],
            "cooking_time": "время в минутах",
            "calories_per_serving": "калории на порцию"
        }
    ]
}
```

#### POST /regenerate_meal
Регенерирует отдельное блюдо.
```json
Request:
{
    "excludedIngredients": ["ингредиент1"],
    "availableIngredients": ["ингредиент2"],
    "requiredIngredients": ["ингредиент3"],
    "mealIndex": "0",
    "mealType": "breakfast"
}

Response:
{
    "meal": {
        "name": "название блюда",
        "ingredients": ["ингредиент - количество"],
        "instructions": ["шаг1", "шаг2"],
        "cooking_time": "время в минутах",
        "calories_per_serving": "калории на порцию"
    }
}
```

### Shopping Routes (app/routes/shopping_routes.py)

#### POST /generate_shopping_list
Генерирует список покупок.
```json
Request:
{
    "meals": [массив рецептов]
}

Response:
{
    "shopping_list": ["ингредиент1 - количество", "ингредиент2 - количество"]
}
```

## 9. Рекомендации по развертыванию

### Требования:
- Python 3.10+
- Anthropic API ключ
- Доступ к интернету (для Claude API)

### Шаги установки:
1. Клонировать репозиторий
2. Создать виртуальное окружение
3. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Настроить переменные окружения:
   ```bash
   ANTHROPIC_API_KEY=your_api_key
   FLASK_ENV=development/production
   ```
5. Запустить приложение:
   ```bash
   python main.py
   ```

### Рекомендации по безопасности:
- Использовать HTTPS
- Защитить API ключи
- Регулярное обновление зависимостей
- Мониторинг логов

### Масштабирование:
- Возможность добавления новых эндпоинтов
- Расширение функционала сервисов
- Добавление новых моделей данных
- Интеграция с базами данных
